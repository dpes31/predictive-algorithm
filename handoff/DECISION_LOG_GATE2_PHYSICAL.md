# Gate 2 Physical Evidence Decision Record

## D-019 — 로또 예측기 유지 및 물리·운영 증거모형 확장

- 결정일: 2026-06-30
- 사용자 의도:
  - 개발 결과는 계속 6개 번호 조합 5세트를 출력한다.
  - 기존 M0~M3와 축적된 알고리즘을 폐기하지 않는다.
  - 로또 예측기는 향후 일반 의사결정 알고리즘으로 확장하기 위한 핵심 실험제품이다.
  - 새로운 관점과 물리변수를 추가해 검증 가능한 예측 가설을 강화한다.

### 결정

- 제품 방향을 일반 의사결정 앱으로 전환하지 않는다.
- 기존 로또 예측기 위에 M4 물리·운영 증거모형을 추가하는 명세를 작성한다.
- 추첨기·볼 세트·교체 구간·사전 운영정보·과거 배출순서 요약을 후보 피처로 정의한다.
- 모든 신규 데이터는 출처·관측시각·사전가용성·신뢰도를 포함한다.
- 현재 회차 배출순서처럼 결과 이후 알 수 있는 값은 입력하지 않는다.
- 기존 M3의 불가능한 검정계약은 단일 maxT omnibus로 교체하는 안을 제안한다.
- 제안 모델 버전은 `3.0.0-research`다.

### 단계

1. Gate 2-3P-1: 명세 — 완료, 사용자 검토 대기
2. Gate 2-3P-2: 구현 — 승인 전 차단
3. Gate 2-3P-3: 합성 재검증 — 구현 전 차단
4. Gate P-1: 실제 메타데이터 파일럿 — 합성검증 전 차단
5. Gate 2-4P: 실제 Walk-forward — 앞 단계 통과 전 차단

### 영향

- 다음 단계는 즉시 재검증이 아니라 먼저 개발이다.
- 개발 전에 M4·데이터·검증계약을 고정한다.
- 합성검증 통과 전 실제 후보 공개와 모바일 UI 개발은 하지 않는다.
- 실패한 2.0.0·2.1.0 연구결과는 그대로 보존한다.
